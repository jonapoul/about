An object representing a single row to be displayed in either an AboutDialogBuilder or a AboutFragment.

Parameters

icon

The Drawable resource ID of an icon to be displayed on the left hand side.

title

The string to be displayed as the row's title.

subtitle

The string to be displayed as the row's subtitle/value.

onClickButton

A nullable callback to be invoked when the user taps on the row. If not null, a "launch" button will be displayed on the right hand side to make the row's clickability more obvious.
